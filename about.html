About This Project

This is a project from Code Crafters, created as a 1st semester project. The purpose of this project is to help students learn the fundamentals of web development, including HTML, CSS, and JavaScript.

By building ShopEase, we practiced how to design user interfaces, manage product data, implement cart functionality, and create a simple checkout flow. The project demonstrates how an e-commerce site works at a basic level, preparing us for more advanced development in future semesters.
